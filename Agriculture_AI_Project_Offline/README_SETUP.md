# Agriculture AI Project - Setup Instructions

## For Your Teacher: How to Run This Project

### Step 1: Extract the Zip File
```
unzip Agriculture_AI_Project_Containerized.zip
```

### Step 2: Navigate to the Project Directory
```
cd Agriculture_AI_Project_Containerized
```

### Step 3: Build and Run the Application
```
docker-compose up --build
```

### Step 4: Access the Application
- **Frontend (Main Interface):** http://localhost:80
- **Backend API:** http://localhost:5000


## Features Included:
 Soil Health Analysis
 Crop Recommendation System
 Fertilizer Planning
 User Authentication
 Responsive Design
